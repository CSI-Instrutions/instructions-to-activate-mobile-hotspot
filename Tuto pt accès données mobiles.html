Instructions pour Modifier le Point d'Accès Mobile

Pour que le service de maintenance puisse se connecter à distance, veuillez suivre les instructions ci-dessous pour modifier le nom de votre point d'accès mobile et le mot de passe. Les étapes sont fournies pour iOS et Android.

Instructions pour iOS

1. Ouvrez les Réglages de Votre iPhone

Ouvrez les réglages de votre iPhone, cliquez sur "Général", puis sur l’option "Informations".

2. Modifiez le Nom de l'Appareil

Sélectionnez la rubrique "Nom" se trouvant tout en haut de la fenêtre et donnez un nouveau nom à votre appareil: "csi".

3. Accédez aux Paramètres de Réseau

Accédez à "Réglages" > "Réseau cellulaires"

4. Activer les données cellulaires

Activer les données cellulaires" > "Partage de connexion" > "Activer le partage de connexion"

5. Modifiez le Mot de Passe

Entrez le nouveau mot de passe du réseau : "csi2024industrie".

5. Enregistrez les Modifications

Assurez-vous que les paramètres sont correctement enregistrés.

Instructions pour Android

1. Accédez aux Paramètres de Réseau

Ouvrez les paramètres de votre téléphone Android et accédez à "Connexions" > "Point d'accès mobile".

2. Activer le point d'accès mobile

Activer le point d'accès mobile en appuyant sur le bouton à droite, puis activer vos données mobiles. Accéder ensuite aux paramètres du point d'accès. > "Point d'accès mobile".

3. Modifiez le Nom du Point d'Accès

Touchez "Configurer le point d'accès Wi-Fi" et modifiez le nom du réseau en "csi" et le mot de passe en "csi2024industrie".

4. Enregistrez les Modifications

Assurez-vous que les paramètres sont correctement enregistrés.

Une fois les modifications effectuées, vous serez prêt à fournir les informations nécessaires au service de maintenance.
